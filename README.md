## MoonJs WebPack ES6 SCSS Boilerplate
Moon.js / Moon.js Router / WebPack / ES6 / SCSS / Now UI Kit

This is simply a example of a moonjs boilerplate with Now UI Kit

`npm install`
- Development: `npm run dev`
- Production: `npm run prod`
